### ✅ Interview-ready answer

> HTTP status codes are sent by the server in the response to indicate the result of a request.  
> They are grouped into ranges:
> 
> - 2xx: Success (e.g. 200 OK)
> - 3xx: Redirection
> - 4xx: Client errors (e.g. 404 Not Found)
> - 5xx: Server errors (e.g. 500 Internal Server Error)
