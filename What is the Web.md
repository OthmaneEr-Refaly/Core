> The World Wide Web is a system that allows clients, usually browsers, to access resources hosted on servers using URLs and the HTTP protocol.  
> It is different from the Internet, which is the physical network infrastructure.  
> The Web works on a client–server model, where servers run continuously and respond to client requests by sending web resources such as HTML pages, images, or API data.
