### ✅ Interview-ready answer

> When a user enters a URL, the browser first resolves the domain name using DNS to get the server’s IP address.  
> Then a TCP connection is established between the client and the server.  
> The client sends an HTTP request, the server processes it and returns an HTTP response containing a status code, headers, and a body.  
> HTTP is stateless, meaning each request is independent, and the browser then renders the response.
